# Yar

## From their [Github](https://github.com/nkanaev/yarr)

Host your own web based RSS feed aggregator `yarr`.

## Server Ports

Ports required to run the server in a table format.

| Port    | default |
|---------|---------|
| App     | 7070    |

### Notes

7070 is the default port, but any port can be used.
